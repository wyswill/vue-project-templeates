## 项目描述

> 该模板基于vue+elementUI进行封装.可以单独下载使用也可以使用手脚架工具 `quicker_cli` 初始化项目

## `quicker_cli` 工具安装方式
```js
npm i -g quicker_cli 
```
## 快速开始

```bash
#下载项目
git clone https://github.com/wyswill/vue-project-templeates.git

#安装依赖
npm install

# 运行调试
npm run dev
```

该项目将会自动在浏览器打开相应地址 http://localhost:9520.
